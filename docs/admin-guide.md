# Admin Usage Guide

## Dashboard

The dashboard highlights district settings, counts of schools, mappings, calendars, and the latest sync activity.

## Schools

Store each school once, along with its MSHSAA identifiers or URLs.

## Sync Mappings

Each mapping represents one publish rule:

- school year
- school
- sport or activity
- level or division
- destination Google Calendar
- enabled state

Mappings can be broad or granular depending on district needs.

## Sync History

Use the history page to confirm when syncs ran and whether they created, updated, skipped, or failed events.

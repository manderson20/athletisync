Google OAuth Configuration

Use the exact same redirect URI in Google Cloud for your OAuth web client.

This URI does not need a public domain. It only needs to be reachable by the browser you use during Google sign-in. If you access AthletiSync at http://172.16.1.77, then the redirect URI can also use http://172.16.1.77.

Get the Client ID and Client Secret from Google Cloud Console: APIs & Services -> Credentials -> Create Credentials -> OAuth client ID -> Web application.

Setup Instructions

Use this section to add the Google account AthletiSync should use for calendar access.

  1. Open https://console.cloud.google.com/.
  2. Create or select a Google Cloud project.
  3. Open APIs & Services -> Library.
  4. Search for Google Calendar API and click Enable.
  5. Open APIs & Services -> Credentials.
  6. Click Create Credentials -> OAuth client ID.
  7. If Google asks you to configure the consent screen first, do that and then come back to the credentials page.
  8. For application type, choose Web application.
  9. Under Authorized redirect URIs, add {{ google_oauth_redirect_uri }}.
  10. After Google creates the OAuth client, copy the Client ID into Google OAuth Client ID on this page.
  11. Copy the Client secret into Google OAuth Client Secret on this page.
  12. Copy {{ google_oauth_redirect_uri }} into Google OAuth Redirect URI on this page if it is not already correct.
  13. Click Save Google OAuth Configuration.
  14. In the form below, enter a connection name such as District Calendar Manager.
  15. Click Sign In With Google.
  16. Sign in with the Google account that already has access to the calendar you want AthletiSync to manage.
  17. Approve the requested Google Calendar access.
  18. After Google sends you back to this page, AthletiSync saves that Google account as an auth profile.
  19. Repeat this section again if you want to connect a second Google account.
  20. Then use Add Calendar to save a separate calendar destination record.
  21. When you add the calendar destination, enter any label you want in Display Name, paste the exact Google Calendar ID, and select the connected auth profile.
  22. Use the Test Calendar Access button in the calendar list to confirm that the connected account has writer or owner access to that specific calendar.

This is where you add the Google account AthletiSync will use. It saves the Google account connection only. You will add the calendar destination separately in the next section.
